# Introduction

Welcome to the **Docs** documentation\!

## Overview

This documentation covers the LearnFlow platform, an AI-powered Python tutoring system.

## Quick Start

```bash
# Start the development server
npm run start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Key Features

- 🤖 AI-powered tutoring agents
- 📝 Interactive code exercises
- 📊 Progress tracking and analytics
- 🎯 Personalized learning paths
